# Akilli-Kutuphane-Yonetim-Sistemi
Öğrenci, personel ve yöneticiler için kitap arama, ödünç talebi ve envanter yönetimini dijitalleştiren, çoklu rol destekli web tabanlı akıllı kütüphane yönetim sistemidir. / A multi-role web-based smart library system digitizing book searches, loan requests, and inventory management for students, staff, and admins.
